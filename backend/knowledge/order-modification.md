# Order modification

Orders can be modified within 1 hour of placement, before they enter the fulfilment queue.

To modify an order, contact support immediately with your order ID and the change you need. We can update quantities, swap product variants (e.g. colour), or change the delivery address.

Once an order has been picked and packed, modifications are no longer possible. You would need to let the order arrive and then start a return if needed.

Order cancellations are free if requested within the 1-hour window. After that, the order may already be in transit and you would need to follow the return process once it arrives.
